# Scripts - Deployment Automation

A collection of bash scripts for initializing and deploying projects in the portfolio.

## Scripts

| Script | Purpose |
|--------|---------|
| `cleanup-git-repo.sh` | Repository cleanup and maintenance |
| `pre-deploy-check.sh` | Pre-deployment validation |
| `setup-all.sh` | Universal project setup |
| `setup-assembly.sh` | Assembly Labs setup |
| `setup-cap.sh` | CAP project setup |
| `setup-cool-curriculum.sh` | Cool Curriculum setup |

## Usage

```bash
# Make script executable
chmod +x script-name.sh

# Run script
./script-name.sh
```

## Tech Stack

- Bash
- npm integration
- Git operations
