## Usage

These personas inform:

1. **Mock data generation** — Dashboard stats, session history
2. **Belt personalization testing** — Switch personas in Settings > Demo Mode
3. **Voice/tone calibration** — Journal entry samples per persona
4. **Risk detection testing** — At-risk personas trigger intervention logic

## The Exhausted User State

Beyond WHO users are, every persona becomes "The Exhausted User" post-training:

- **Physical:** Heart rate 100-140 BPM, sweating, grip depleted
- **Cognitive:** 40% impaired, 30-60 second attention span
- **Emotional:** High or low based on session, low tolerance for friction

See `/research/USER_PERSONAS_AND_RESEARCH.md#the-exhausted-user` for full details.
