## Key Concepts

### Belt Progression Timeline (IBJJF Minimums)

| Transition | Minimum Time |
|------------|--------------|
| White → Blue | No minimum (typically 1-2 years) |
| Blue → Purple | 2 years |
| Purple → Brown | 18 months |
| Brown → Black | 1 year |

### Position Categories

1. **Guard** — Closed, open, half, butterfly, spider, de la riva, x-guard
2. **Top** — Mount, side control, knee on belly, back control
3. **Escapes** — From mount, side control, back, turtle
4. **Takedowns** — Wrestling, judo, clinch
5. **Leg Locks** — Ashi garami positions, legal submissions by belt
