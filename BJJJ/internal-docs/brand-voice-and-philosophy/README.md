## Core Philosophy

> **"Every practitioner deserves to see their journey clearly."**

We don't replace coaches—we amplify them. We don't gamify the grind—we illuminate it.

---

## Brand Archetype

**The Dedicated Training Partner**

We're the purple belt who's been training for six years. Not the loudest person in the room, but the one everyone trusts. We remember what it felt like to be new. We're still learning too.

---

## Five Pillars

1. **Understanding beats memorization** — Show how techniques connect
2. **Fundamentals are advanced techniques** — Celebrate basics mastery
3. **Every journey is valid** — Competitors and hobbyists alike
4. **Progress is personal** — Measure growth, not rank against others
5. **Consistency compounds** — Reward the work, not just wins

---

## Voice in Five Words

**Knowledgeable. Warm. Direct. Grounded. Supportive.**

---

## Non-Negotiables

- Never gamify belt progression
- Never contradict a user's coach
- Never promise faster promotions
- Never use empty hype language
- Never disrespect the tradition

## Instructor Influence

The `instructor-influence-matrix.md` documents 10 elite BJJ instructors and their teaching philosophies. This matrix informs:

- **App feature context** — Which pedagogy matches which feature
- **AI assistant responses** — Cite instructors when relevant
- **Content creation** — Match voice to use-case

### Pedagogy Types

| Type | Key Instructors | App Application |
|------|-----------------|-----------------|
| Systems-Based | Danaher, O'Conaill | Technique relationships, position flows |
| Fundamentals-First | Roger Gracie, Gurgel | Curriculum scaffolding, belt progress |
| High-Performance | Galvao, Vieira | Competition prep, analytics |
| Technical Precision | Musumeci, Bastos | Detailed notes, proficiency stages |
| Motivational | DeBlass | Recovery, injury, plateau messaging |
| Adaptive | Renzo Gracie | Balance tradition and evolution |
