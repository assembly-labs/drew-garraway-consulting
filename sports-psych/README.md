# Sports Psych - Daily Wisdom

A daily sports wisdom app displaying motivational quotes tailored to sports psychology, with simplified explanations for athletes.

## Features

- Daily rotating quotes (365+ quotes)
- Simplified explanations of each quote
- Sports psychology focused
- Clean, motivational UI

## Tech Stack

- Vanilla JavaScript
- HTML5
- CSS3

## Files

- `index.html` - Main page
- `script.js` - Quote cycling logic
- `styles.css` - Styling
- `quotes-data.js` - Quote database (365+ entries)

## Usage

Open `index.html` in a browser. Quote changes based on day of year.
