# Accountable - Habit Tracker

A habit-tracking web app that helps users build and maintain accountability for daily habits with tracking and analytics.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Zustand (state management)
- date-fns
- Framer Motion

## Getting Started

```bash
cd habit-tracker
npm install
npm run dev
```

## Deployment

Deployed to Cloudflare Pages. Use the deployment script:

```bash
./deploy-habit-tracker.sh
```

## Status

Functional and actively maintained.
