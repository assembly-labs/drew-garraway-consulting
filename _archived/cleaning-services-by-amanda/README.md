# Cleaning Services by Amanda

Professional website for a residential cleaning service in the Main Line and King of Prussia areas.

## Features

- Service tier descriptions
- Appointment booking
- Contact information
- Mobile-responsive design

## Tech Stack

- HTML5
- CSS3 (inline with CSS variables)
- No build tools required

## Structure

Single-page static website (`index.html`) with all content, styling, and scripts inline.

## Deployment

Static hosting - simply serve `index.html`.

## Status

Production-ready.
