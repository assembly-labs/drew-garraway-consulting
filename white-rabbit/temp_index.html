Kaijutsu

\

A cooperative card game where building emotional intelligence looks like battling monsters together.

\

Children learn to recognize their emotions, navigate conflict, and work as allies\'97not through worksheets or lectures, but through genuine play. The kind where struggle is part of the adventure, not something to avoid.

\

Because emotional intelligence isn't a soft skill. It's foundational infrastructure. Everything else builds on it.

Librarian LLM

\

What if you could talk to a library the way you talk to a brilliant, endlessly patient librarian?

\

Not keyword searches that return lists. Real conversation. Natural questions that lead to genuine discovery.

\

Librarian LLM creates a conversational interface for exploring library collections\'97helping educators, students, and families navigate materials with curiosity instead of frustration.

\

Because finding the right resource shouldn't feel like searching for a needle in a haystack. It should feel like following a fascinating thread.

Cool Curriculum

\

Thoughtful lesson planning shouldn't take hours. But it should take thought.

\

Cool Curriculum helps educators and parents create engaging lesson plans, activities, and assessments\'97not by automating away the thinking, but by accelerating the parts that don't require expertise so you can focus on the parts that do.

\

AI as a tool for educators, not a replacement. Because nobody knows your students like you do.

\

Intentional technology serving humans, not metrics.
