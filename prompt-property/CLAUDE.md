# Prompt Property - Claude Context

## What This Is

AI assistant for property managers and residents. Provides instant answers to FAQs (pool rules, pet policies, quiet hours, lease questions).

## Tech Stack

- **Framework:** Vanilla JavaScript
- **Styling:** CSS3
- **Deployment:** Static HTML (Vercel)

## Key Files

- `index.html` - Landing page
- `demo.html` - Interactive demo
- `app.js` - Application logic
- `styles.css` - Styling

## Commands

No build step. Open HTML files directly in browser.

## Key Patterns

- Document-based knowledge system
- Multi-property support planned
- Static site, no backend required for demo
