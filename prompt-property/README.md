# Prompt Property - Property Management Assistant

An AI assistant for property managers and residents. Provides instant answers to FAQs (pool rules, pet policies, quiet hours, lease questions) to save managers 20-30% of their time.

## Features

- Instant FAQ answers
- Multi-property support
- Document-based knowledge system
- Resident self-service portal

## Tech Stack

- Vanilla JavaScript
- HTML5
- CSS3
- Document-based knowledge system

## Files

- `index.html` - Landing page
- `demo.html` / `demos.html` - Demo versions
- `app.js` - Main application logic (37KB)
- `styles.css` - Styling

## Usage

Open `index.html` in a browser for the landing page, or `demo.html` for the interactive demo.

## Status

Complete prototype/MVP (demo-ready).
