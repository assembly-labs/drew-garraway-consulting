# Raw Notes Directory

This folder contains **Phase 2 output**: factual extraction from screenshots.

## Rules for Raw Notes

1. **NO brand voice** - Just facts, figures, formulas
2. **Complete extraction** - Capture everything from the source
3. **Clear formatting** - Use headers, bullets, tables
4. **Source tracking** - Note which screenshot(s) the content came from
