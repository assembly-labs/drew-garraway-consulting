## Content Pipeline

### Ideas → Draft → Review → Publish

1. **Ideas Bank** (`/content/ideas.md`)
   - Capture all content ideas
   - Rate by impact/effort
   - Tag with category

2. **Drafts** (`/content/drafts/`)
   - Work in progress
   - Include research notes
   - Fact-check claims

3. **Review Checklist**:
   - [ ] Facts verified
   - [ ] Voice consistent
   - [ ] Punk level appropriate
   - [ ] Value delivered
   - [ ] Call-to-action clear

4. **Publishing**:
   - Generate HTML from Markdown
   - Update navigation/index
   - Share on channels

## Content Style Guide

### Writing Rules:

1. **Be Real**: Use actual numbers
2. **Be Raw**: Don't sugarcoat failures
3. **Be Rebellious**: Challenge conventions
4. **Be Helpful**: Always provide value
5. **Be Yourself**: Drew's voice always

### Language Guidelines:

- Profanity: Strategic, not gratuitous
- Jargon: Explain or avoid
- Tone: Confident, not arrogant
- Length: As long as valuable

## Legal Considerations

### Disclaimers:

- Not financial advice
- Personal experience only
- Results not typical
- Do your own research

### Privacy:

- Anonymize personal data
- No real account numbers
- Percentages over absolutes when sensitive
