# Calc - Cost Calculator

A canned product cost calculator for comparing bids and calculating per-unit costs across product variants.

## Features

- Compare multiple product bids
- Calculate per-unit costs
- Sortable results table
- CSV export

## Tech Stack

- Vanilla JavaScript
- HTML5
- CSS3

## Files

- `index.html` - Main page
- `script.js` - Calculator logic
- `styles.css` - Styling
- `config.js` - DOM helpers and render functions

## Usage

Open `index.html` in a browser. No build step required.

## Status

Production-ready.
