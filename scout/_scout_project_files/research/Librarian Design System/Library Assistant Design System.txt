ğŸ”§ Integration Guide
With React/Vue/Angular
Import the CSS globally:
import './design-system.css';

Use class names in your components:
<div className="message message--assistant">
  <p>{messageText}</p>
</div>

With Backend API
Use the JavaScript utility functions:
// When user sends message
const userMessage = textarea.value;
addMessage(userMessage, "user");
showTypingIndicator();

// Call your API
const response = await fetch('/api/chat', {
  method: 'POST',
  body: JSON.stringify({ message: userMessage })
});

const data = await response.json();
hideTypingIndicator();
addMessage(data.response, "assistant");

// If response includes books
data.books.forEach(book => addBookCard(book));


ğŸ¯ Best Practices
Do's âœ…
Use semantic HTML (<main>, <nav>, <article>)
Keep messages concise (3-4 bubbles max)
End assistant turns with questions or CTAs
Provide clear error recovery paths
Test with keyboard navigation
Test with screen readers
Don'ts âŒ
Don't override design tokens without good reason
Don't create walls of text (respect conversation design)
Don't forget alt text on images
Don't use color alone to convey information
Don't create accessibility barriers

ğŸ› Troubleshooting
Q: Themes not switching?
Check data-theme attribute on <html> element
Verify app.js is loaded after DOM
Q: Buttons not clickable?
Ensure app.js is included
Check browser console for JavaScript errors
Q: Styles not applying?
Verify design-system.css path is correct
Check that CSS loads before page render
Clear browser cache
Q: Menu won't open?
Verify IDs match: menu-toggle, side-menu, menu-overlay
Check that app.js is loaded
