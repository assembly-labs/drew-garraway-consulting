# Download the Chester County Library App

1. Make sure the patron's device is connected to the library's Wi-Fi network.

2. Have them go to the App Store or Play Store on their device.

3. Search for "Chester County Library System".

4. Click "Download and Install" - The icon should look similar to this:
   [Chester County Library System logo]

5. Click "Manage Accounts"

6. Enter the Library Card Number and PIN.

   If the patron does not have a PIN, please see directions titled "Creating Patron PIN" in the 3-Ring Binder.

7. Once the system has verified the patron's information, you can give a brief tour of the App and highlight the following if time permits:
   - **"Digital Library Card"** – The patron's phone/device can now be their library card.
   - **"My Checkouts"** – If the patron has books checked out, you can point out the due dates.
   - **"Renew Selected"** by clicking the book, also be renewed.
   - **"My Holds"** – This is a list of items that the patron currently has on hold.
   - **"My Plan"** – Using the "Manage Settings" button.

8. There are many other helpful features on the App! Please feel free to ask Jennifer if you have questions.
