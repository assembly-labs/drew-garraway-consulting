## Creating a PIN
## (Personal Identification Number)

1. Go to https://catalog.ccls.org/patroninfo or scan this QR Code:
   [QR Code image]

2. Enter your library card number in this field:
   [Screenshot showing library card number entry field]

3. Leave the PIN field blank and click submit.

4. You will be asked to enter their new PIN two times*. See below:
   [Login screen showing PIN entry fields]

5. Use this PIN to finish signing into the Mobile App!

*Please NOTE - You can also use this website to set a new PIN if you do not remember it. To do this, click "Forgot your PIN?" after entering your barcode.

*It is recommend you use a four digit number. PIN may not include numbers which repeat three times, example: 8777 nor numbers which repeat two digits, example: 1212.
