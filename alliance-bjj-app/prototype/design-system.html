Animation Best Practices

  • Performance: Use GPU-accelerated properties (transform, opacity)
  • Accessibility: Respect prefers-reduced-motion media query
  • Timing: Keep animations under 400ms for optimal perception
  • Easing: Use cubic-bezier(0.16, 1, 0.3, 1) for smooth, natural motion
  • Staggering: Add 50-100ms delays between list items

Accessibility Checklist

  • ‚úÖ All text meets WCAG AA contrast ratios (4.5:1 normal, 3:1 large)
  • ‚úÖ Focus indicators visible on all interactive elements
  • ‚úÖ Animations respect user motion preferences
  • ‚úÖ Color not used as sole indicator of state
  • ‚úÖ Keyboard navigation fully functional
  • ‚úÖ Screen reader friendly markup
