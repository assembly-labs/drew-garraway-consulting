  • Deliverables: Design system, all screens, prototypes

  • Backend Focus: Node.js/PostgreSQL

Key Development Considerations

Critical Path

  • Shopping cart & checkout (core revenue)
  • Order management for vendors
  • Pick lists for market staff
  • Mobile-responsive design

Quick Wins

  • Start with Berwyn data (70+ vendors)
  • AI-generate product descriptions
  • Use existing design system
  • Leverage Stripe Connect

Risks to Mitigate

  • Vendor photo quality (provide guidelines)
  • Inventory accuracy (real-time sync)
  • Payment processing (test thoroughly)
  • Mobile performance (optimize images)

  • Browse-to-buy: >3%
  • Cart abandonment: <30%
