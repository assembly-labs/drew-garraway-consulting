# Gather - Claude Context

## What This Is

Farmers market platform connecting buyers with vendors. Supports 8,140+ markets with multi-vendor shopping, SNAP/EBT payments, and market manager tools.

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS + shadcn/ui
- **State:** Zustand + React Hook Form
- **Database:** PostgreSQL + Prisma
- **Payments:** Stripe Connect
- **Deployment:** Vercel + AWS

## Key Files

- `index.html` - Landing/marketing page
- `sitemap.html` - Site structure
- `ROADMAP.md` - 8-month development plan

## Commands

| Command | Purpose |
|---------|---------|
| `npm install` | Install dependencies |
| `npm run dev` | Start dev server |
