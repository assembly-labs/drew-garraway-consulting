# TOMO Voice

Voice-first BJJ transcription app - companion to the main TOMO training journal.

## Features

- Voice recording and transcription
- Session notes via speech
- Integration with TOMO main app

## Tech Stack

- React 19 + TypeScript
- Vite
- CSS Variables

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Auto-deploys to Cloudflare Pages on push to `main`.
