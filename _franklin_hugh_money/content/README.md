## Content Types

### SIE Exam (`/sie-exam/`)
Study materials for the Securities Industry Essentials exam.
- **raw-notes/**: Markdown files with raw study notes from textbooks, screenshots, etc.
- **study-guides/**: Polished content ready for the website
- **practice/**: Practice questions, flashcards, and test prep materials

### Blog (`/blog/`)
Personal finance content written in Franklin Hugh's voice.
- **journal/**: Monthly portfolio updates and reflections
- **lessons/**: Investment lessons learned
- **rants/**: Commentary on financial industry practices

### Tools (`/tools/`)
Interactive financial calculators and planning tools.

### Resources (`/resources/`)
Curated lists of books, websites, and other helpful resources.

## Content Guidelines

### File Naming
- Use lowercase with hyphens: `chapter-05-municipal-debt.md`
- Include chapter numbers with leading zeros: `chapter-01.md`, not `chapter-1.md`
- Date journal entries: `2024-12-portfolio-update.md`

### Markdown Format
All content should be in Markdown format with:
- Clear heading hierarchy (# for title, ## for main sections)
- Descriptive frontmatter when needed
- Consistent formatting throughout

### Organization
- Keep raw notes separate from processed content
- One file per chapter/topic for easy management
- Use subdirectories for complex topics if needed

## Adding New Content

1. **SIE Chapters**: Add raw notes to `sie-exam/raw-notes/`, then create polished version for website
2. **Blog Posts**: Add directly to appropriate blog subfolder
3. **Tools**: Add documentation to `/tools/` folder
4. **Resources**: Update resource lists in `/resources/`
