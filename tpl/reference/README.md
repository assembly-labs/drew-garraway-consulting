# Toshiba e-STUDIO 2525AC Documentation

This folder contains manuals, user guides, quick start guides, and other documentation for the **Toshiba e-STUDIO 2525AC** multifunction color printer.

## Device Overview

The Toshiba e-STUDIO 2525AC is a compact A3 color multifunction printer (MFP) designed for hybrid work environments. Key specifications:

- **Print Speed**: 25 pages per minute (color and B&W)
- **Display**: 10.1" tablet-like touchscreen
- **Functions**: Print, Copy, Scan, Fax (optional)
- **Paper Capacity**: 1,200 to 5,200 sheets (depending on configuration)
- **Paper Size**: Up to 11" x 17" (A3)
- **Security**: TPM 2.0 encryption, Common Criteria certification

## Common Tasks Quick Reference

### Copying a Document
1. Place original on glass or in document feeder
2. Select **Copy** on the touch panel
3. Adjust settings (color, duplex, paper size)
4. Press **Start**

### Scanning to Email
1. Place original on glass or in document feeder
2. Select **Scan** > **Email**
3. Enter recipient email address
4. Adjust scan settings as needed
5. Press **Start**

### Printing from Computer
1. Open document and select **Print**
2. Choose **TOSHIBA e-STUDIO2525AC** as printer
3. Adjust settings (color, duplex, paper size)
4. Click **Print**

### Accessing TopAccess (Web Administration)
1. Open web browser
2. Enter printer's IP address in address bar
3. Log in with administrator credentials
4. Configure settings as needed
