About This Resource

These guides help both reference desk staff and patrons use the copier independently. During busy periods, patrons can follow these instructions to complete printing, scanning, and copying tasks without waiting for staff assistance.

This documentation is a work in progress. As staff encounter copier quirks or frequently asked questions, those details will be added here so everyone has a consistent reference.
