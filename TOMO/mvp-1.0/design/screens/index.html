Workflows

How screens connect to each other and what triggers transitions.

1. First Launch (Onboarding)

Auth Screen→Welcomesuccessful sign-up
Welcome→About Youtap "Get Started"
About You→Your Trainingtap "Continue" (all fields filled)
Your Training→Get Startedtap "Continue" (gym selected)
Get Started→Tab Bar (Journal)tap "Start Training" (after chat payoff)
Auth Screen→Tab Bar (Journal)sign-in (existing user)

2. Log a Session (Voice Path)

Tab Bar Log Button→Entry Phasetap gold + button
Entry Phase→Recording Phasetap "Record Session"
Recording Phase→Processing Phasetap "Stop Recording"
Processing Phase→Review Phasepipeline completes (or timeout/skip)
Review Phase→Success Phasetap "Save Session"
Success Phase→Journal Listauto-dismiss after 4 sec (or double-tap)
Entry Phase→Review Phasetap "Just type it" (skips recording)
Voice preference user→Recording Phaseauto-start (skips Entry)

3. Browse Journal

Journal List→Session Detailtap a session card
Session Detail→Journal Listtap back button

4. Edit a Saved Session

Session Detail→Edit Sheet (modal)tap any editable section
Edit Sheet→Session Detailtap Save or Cancel

5. Delete a Session

Session Detail→Confirmation Alerttap "Delete this session"
Confirmation Alert→Journal Listtap "Delete"

6. Switch Gym (Temporary)

Gym Chip→Gym Picker Sheettap chip on Entry or Review
Gym Picker Sheet→Gym Chip (gold override)search + select gym
Gym Chip (X button)→Gym Chip (home gym)tap X to reset

7. Profile Updates

Profile Screen→Edit Sheet / Pickertap any row
Edit Sheet→Profile Screensave changes

8. Sign Out

Profile Screen→Confirmation Alerttap "Sign Out"
Confirmation Alert→Auth Screenconfirm sign out

Welcome Screen

Onboarding 1/4 · WelcomeScreen.tsx

First screen new users see after signing up.

Elements

Progress Bar
Thin bar at top showing 1/4 progress. Auto-adjusts if screens are added/removed.
App Logo
"TOMO" in large white text (48px). "友" kanji below in gold (32px). Centered on screen.
Typewriter Tagline
Cycles through 3 phrases: "Your training. Remembered." / "Every session. A lesson." / "Let's roll." Gold blinking cursor. Tap to skip animation.
Get Started Button
Full-width gold button at bottom. Black text. Medium haptic on tap. Navigates to About You.

About You Screen

Onboarding 2/4 · AboutYouScreen.tsx

Collects identity info: name, belt, stripes, age, gender.

Elements

Back Button
Top-left arrow, goes back to Welcome.
Progress Bar
Shows 2/4 progress.
Screen Title
"About You" in white (28px).
Name Field REQUIRED
Label: "FIRST NAME". Text input, dark background. Placeholder: "Your first name".
Belt Picker REQUIRED
5 colored circles: White, Blue, Purple, Brown, Black. Selected gets gold border + gold background. Light haptic on tap.
Stripes Picker
5 square buttons: 0, 1, 2, 3, 4. Selected gets gold background. Only appears after belt is picked. Defaults to 0.
Birthday Picker REQUIRED
Tap opens date spinner. Shows "X years old" in gold. Must be 18+. Red error if under 18.
Gender Picker REQUIRED
Two pills: "Male" and "Female". Selected gets gold background + gold text.
Continue Button
Full-width. Gold when all required fields filled, gray when incomplete. Navigates to Your Training.

Your Training Screen

Onboarding 3/4 · YourTrainingScreen.tsx

Collects gym, training frequency, and optional goals/experience/weight.

Elements

Location Permission Card
Animated card. Map pin icon + "Find gyms near you". Enable Location (gold) / Skip (gray outline) buttons. Fades out after choice.
Gym Search Input REQUIRED
Text input with autocomplete dropdown (2+ chars). Shows gym matches with city/state. Unmatched text = custom gym. Selected shows gold checkmark.
Nearby Gyms List
Cards with gym name, distance, city, state. Only shows if location was granted. Tap to select.
Training Target Picker
Three pills: "1-2x / week", "3-4x / week" (default), "5+ / week". Selected gets gold highlight.
Training Goals OPTIONAL
Multi-select chips: Competition, Fitness, Self-Defense, Mental Health, Community, Hobby. Tap to toggle.
Experience Level OPTIONAL
Four options: < 6 months, 6mo-2yr, 2-5yr, 5+ yr.
Body Weight OPTIONAL
Number input + lb/kg toggle. Shows competition weight class based on gender.
Continue Button
Enabled once gym is selected. Navigates to Get Started.

Get Started Screen

Onboarding 4/4 · GetStartedScreen.tsx · Can't swipe back

Final onboarding. Logging preference, mic permission, welcome chat.

Elements

Screen Title
"Ready to get rolling?"
Voice First Button
Primary gold button. Selects voice logging, triggers mic permission flow.
Type It In Button
Secondary outline button. Selects text logging, skips mic permission.
Mic Permission Primer CONDITIONAL
Modal explaining mic access (voice path only). "Allow" (blue) and "Use Text Instead" (gray) buttons.
Chat Payoff Overlay
Full-screen. TOMO avatar (gold circle, "友"). 5 sequential chat bubbles with typewriter animation. Typing indicator (3 pulsing dots) between bubbles. Personalizes with name, belt, gym.
Start Training Button
Gold button, appears after all chat bubbles finish. Enters the main app.

Entry Phase

Session Logging 1/5 · SessionLoggerScreen.tsx · Skipped for voice-preference users

Pre-recording form where you set up your session before recording.

Elements

Cancel Button
Top-left text link. Navigates to Journal List. No confirmation needed.
Screen Title
"Log Your Training" (28px, white).
Gym Chip
Pill button with map pin + gym name. Tap to open Gym Picker Sheet. Gold border if overridden.
Training Mode Chips
Label: "TRAINING MODE". Three options: Gi, No-Gi, Other. Large chips (56px). Gold when selected.
Session Kind Chips
Label: "SESSION KIND". Four options: Regular Class, Open Mat, Comp Training, Other.
Duration Chips
Label: "DURATION". Six options: 30, 45, 60, 75, 90, 120 min. Square chips (44x44). Default: 90.
Spar Toggle
Label: "DID YOU SPAR?" Two chips: Yes, No. No default.
Record Session Button
Large gold button. Starts mic recording, moves to Recording Phase. Medium haptic.
Just Type It Link
Text link below record button. Skips to Review Phase with empty transcript.
Recording Prompt
Random prompt above the record button. "What did you work on today?" / "How was training?" etc.

Recording Phase

Session Logging 2/5 · SessionLoggerScreen.tsx

Live voice recording. User talks about their training session.

Elements

Cancel Button
Top-left text link. Short recordings cancel immediately. Longer recordings show confirmation: "Keep Recording" / "Discard".
Gym Label
Read-only. Map pin icon + gym name. Small gray text, not tappable.
Pulsing Circle
120x120 circle. Gold border (3px), transparent fill. "友" kanji inside (32px, gold). Pulses with opacity animation.
Session Timer
Elapsed time display (e.g., "2:34"). 48px white text, extra-bold font.
Recording Prompt
Rotating text below timer. Cycles through encouraging questions every few seconds.
Stop Recording Button
Gold button. Stop icon (black) + "Stop Recording" (black text). Moves to Processing Phase.

Processing Phase

Session Logging 3/5 · SessionLoggerScreen.tsx

Loading screen while voice is transcribed and analyzed. 150-second timeout.

Elements

Loading Spinner
Large gold activity indicator, centered.
Status Text
Changes as pipeline progresses: "Transcribing..." then "Analyzing..." then "Almost there..."
Skip / Cancel Option
Skips to Review Phase with empty extraction. User fills everything manually.

Review Phase

Session Logging 4/5 · SessionLoggerScreen.tsx

Editable form showing extracted data. Fix anything wrong, fill in what's missing, then save.

Header

Cancel Button
Left side. Discards everything, back to Journal List.
Review Title
"Review Session" centered (17px, white).

Form Fields

Gym Chip
Pill with map pin + gym name. Tap to open Gym Picker Sheet. Gold border if overridden.
Detail Badges Row
Two tappable badges: Mode Badge ("Gi"/"No-Gi"/"Other" + edit icon) and Duration Badge ("90 min" + edit icon). Tap to expand inline editor.
Mode Editor CONDITIONAL
Opens on Mode Badge tap. Three chips: Gi, No-Gi, Other. Closes on selection.
Duration Editor CONDITIONAL
Opens on Duration Badge tap. Six chips: 30, 45, 60, 75, 90, 120. Closes on selection.
Warm-up Field
Label: "WARM-UP". Yes/No chips. Gold left border + gold tint if unanswered.
Techniques Field
Label: "TECHNIQUES TRAINED / LESSON TOPIC". Removable pill tags + text input with autocomplete dropdown. "recent" badge on previously-logged techniques. Gold border if empty.
Did You Spar Field
Label: "DID YOU SPAR?" Yes/No chips. "No" hides sparring sections below.
Sparring Rounds IF SPARRING
Label: "SPARRING ROUNDS". Seven chips: 1-7+. Gold border if none selected.
Submissions Landed GREEN IF SPARRING
Label: "SUBMISSIONS LANDED". Green tags with +/- count buttons. Long-press for jiggle-delete mode. Text input with autocomplete.
Got Caught RED IF SPARRING
Same pattern as Submissions Landed but red-tinted. Tracks submissions received.
Injuries Field ORANGE
Label: "INJURIES". Orange tags, removable. Text input. Gold border if empty.
Instructor Field
Label: "INSTRUCTOR". Text input: "Who taught class?" Gold border if empty.
Session Type Chips
Label: "SESSION TYPE". Four chips: Regular Class, Open Mat, Comp Training, Other.
Voice Transcript IF VOICE
Label: "VOICE TRANSCRIPT". Raw recording text, italic gray, max 200px scroll. Bottom of form.
Save Session Button
Full-width gold. "Save Session" (black text). Spinner while saving. On success: moves to Success Phase.

Success Phase

Session Logging 5/5 · SessionLoggerScreen.tsx + PostSaveQuote.tsx

Elements

Post-Save Quote
Full-screen overlay. Belt-aware martial arts quote for 4 seconds. Progress bar at bottom. Double-tap to dismiss early. Then navigates to Journal List. See Post-Save Quote component below.

Post-Save Quote

Quote Attribution
Below quote. Monospace, 12px, gold at 60% opacity. Person's name.
