Opener Variety

Each week leads with whatever is most noteworthy -- a milestone, a technique observation, a consistency note, or a direct response to their own words. Never the same structure twice in a row. Never starts with their name unless it's the first insight they've ever received.

Output Schema (Simplified)

{ paragraphs: [
    { text: "...", isWatch: false },
    { text: "...", isWatch: false },
    { text: "...", isWatch: true, defer: "Talk to your coach about..." }
  ],
  focusNext: "..." }

Simple array of paragraphs. Each is just text. The only visual differentiation is `isWatch: true` which gets the red left border. Everything else renders as plain message text. Bold is applied inline via markdown-style markers in the text. The focus recommendation is always last, separated by a line.
