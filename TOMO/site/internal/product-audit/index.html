Core

Voice Recording + Auto-Stop

Pulsing gold mic with live timer. Auto-stops at 90 seconds because users are exhausted post-training. AAC format via expo-audio. Text-only fallback always available.

AI Extraction

13 Structured Fields From Voice

Claude Haiku parses messy voice transcripts into structured data. Preserves the practitioner's own technique phrasing.

  • Training mode (gi/nogi), session kind (class/open mat/comp training)
  • Duration, warm-up, sparring rounds
  • Techniques drilled (array), submissions given/received (with counts)
  • Injuries, instructor name, lesson topic
  • Clean 1-3 sentence journal summary

03 / Insights

AI Training Debriefs

The differentiating feature. TOMO generates personalized weekly training debriefs using a three-tier AI architecture with context cascade. 10 enhancements shipped.

Three-Tier Architecture: Weekly debriefs (Haiku, "training partner" persona) feed into monthly reviews (Sonnet, "coach" persona) which feed into quarterly assessments (Opus, "black belt" persona). Each tier passes context downward so insights stay coherent across timescales.

Intelligence

Belt-Adaptive Prompting

Every AI prompt dynamically adjusts tone, vocabulary depth, and encouragement based on the user's belt level.

  • White belt: warm, supportive, celebrate showing up, basic BJJ terms
  • Blue belt: coaching, balanced, identity-building, intermediate terms
  • Purple belt: collaborative, direct, analytical, advanced terms
  • Brown/Black: peer, professional, systems-level, expert terms

10 Shipped Enhancements (ENH-01 through ENH-10)

ENH-01The Almost Knew
Leads with highest-surprise observation, not session count. The first sentence should make users stop scrolling.
ENH-02Focus Echo
Shows last week's focus recommendation at the top. AI checks if the user followed through.
ENH-03The Angle Shift
Rotates through 6 analytical lenses weekly (technique, rhythm, energy, partnerships, positions, recovery). Never repeats.
ENH-04The Teaser
Post-save success phase previews the weekly debrief. 6 rotating templates, zero AI cost.
ENH-05Mood Pulse
Optional 5-dot mood rating after save. Injected into weekly prompt for mood-training correlation analysis.
ENH-06The Quiet Week
Warm re-entry for users returning after 14+ day gap. No guilt, just: "The mat will be there."
ENH-07The Reframe
Detects plateaus (4+ weeks flat) and reframes as unseen progress. Never says "plateau" or "stuck."
ENH-08One Year Ago
Resurfaces sessions from 3/6/12 months ago. AI generates comparison showing how training evolved.
ENH-09The Thread
Technique mentions are tappable gold text. Reveals contributing sessions with excerpts. Cross-tab navigation.
ENH-10Tell Me More
Gold link below each insight. One-exchange chat follow-up via dedicated edge function. Response persists.

Security

Row Level Security + Private Audio

Every table has RLS policies. Users can only access their own data. Audio files stored in a private bucket with 15-minute signed URLs. API keys never ship in the app binary. 18+ age enforcement via database trigger.

Not Collecting (But Users Already Describe)

The 90-Second Paradox: Users already describe far more than we index. The transcript contains training intent, partner names, body state, coaching feedback, technique failures, and emotional arcs -- all of which are lost after extraction. The highest-leverage next move isn't collecting new data; it's indexing data users are already giving us.

Lost Data Example
Training intent"I wanted to focus on half guard today"
Partner context"Rolled with Marcus (blue) and two whites"
Body state"Still sore from yesterday, low energy"
Failed attempts"Tried omoplata twice but couldn't finish"
Coach feedback"Coach said tighten posture on the pass"
Session arc"Started frustrated, breakthrough in the last round"
Recovery"Iced my knee after class"
08 / Biggest Opportunities

What Would Maximize Insights Value

Ranked by impact-to-effort ratio. Tier 1 items unlock the most value with the least friction.

Tier 1 0-2 SPRINTS
Enhanced Transcript Extraction
Extract training intent, partner names, body state, failed technique attempts, and coaching feedback from transcripts users are already recording. Zero new user behavior required.
Unlocks: intent vs. outcome analysis, partner graph, fatigue-adjusted insights
Tier 1 0-2 SPRINTS
Technique Proficiency Rating
One tap per technique in review phase: Learning / Developing / Solid. Distinguishes "drilled" from "owned" and tracks confidence growth over time.
Unlocks: "Armbar went from Learning to Solid in 6 weeks" insight
Tier 2 2-4 SPRINTS
Reflections Feature (FEAT-010)
Post-session voice/text thoughts, separate from structured data. "What I learned" creates a retention moat and reveals coaching intent.
Unlocks: connects what happened to what user internalized
Tier 2 2-4 SPRINTS
Persistent Injury Tracking (FEAT-006)
Structured injury log: body part, severity, duration, recovery attempts. Essential for risk detection and recovery-sensitive insights.
Unlocks: "Left knee for 4th week + heel hook volume = pattern?"
Tier 3 4+ SPRINTS
Competition Tracker (FEAT-005)
Pre-comp weight, bracket results, match records. Tournament outcomes are the ultimate performance metric.
Unlocks: competition readiness, training-to-comp performance gap
Tier 4 8+ SPRINTS
Coach Integration (FEAT-007)
Coaches see student roster + logs + add feedback. Closes the loop: "Coach said focus on X" → student logs sessions → insights reference coach feedback.
Unlocks: coach-integrated feedback loop, institutional adoption

10 / Bottom Line

What we built: A voice-first BJJ journal that turns a 90-second voice memo into a structured training log, then delivers belt-personalized weekly AI debriefs that notice patterns the practitioner can't see from inside their own training.

What makes it defensible: The three-tier AI architecture with context cascade creates insights that get smarter over time. Belt-adaptive prompting means a white belt and a black belt get fundamentally different experiences. Cost per user is ~$1.40-2.20/year.

Biggest unlock ahead: Users already describe far more than we index. Enhanced extraction from existing transcripts is the lowest-friction path to dramatically richer insights -- no new user behavior required.
