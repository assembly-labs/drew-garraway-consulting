# TOMO - BJJ Training Journal

A voice-first training journal for Brazilian Jiu-Jitsu practitioners to log sessions, track belt progression, and build a technique library.

## Features

- Voice-first session logging (capture insights post-training)
- Belt progression tracking
- Technique library with positions and submissions
- Session analytics and trends

## Tech Stack

- React 18 + TypeScript
- Vite
- CSS Variables
- localStorage (prototype phase)

## Getting Started

```bash
cd prototype
npm install
npm run dev
```
