## Key Decisions

1. **Mobile-first** — Practitioner app is primary, coach/owner are secondary
2. **Voice-first logging** — Forms are fallback, not primary
3. **Belt personalization** — Every screen adapts to belt level
4. **60-second flows** — All primary actions complete in under a minute
