## Sourcing Workflow

### Step 1: Find Video
1. Search YouTube for technique + instructor
2. Verify video quality (720p+, clear audio)
3. Check upload date (prefer <5 years old)
4. Confirm technique accuracy

### Step 2: Extract Metadata
```
youtube_id: [11-character ID from URL]
instructor: [Full name]
title: [Exact title]
duration_seconds: [Convert MM:SS to seconds]
```

### Step 3: Add Enrichment
```
difficulty_level: [1-10]
belt_prerequisite: [white|blue|purple|brown|black]
gi_nogi: [gi|nogi|both]
concepts: [tag1|tag2|tag3]
quality_score: [1-5]
```

### Step 4: Add to Database
1. Add row to `video_enrichment_template.csv`
2. Update `techniqueVideos.ts` with new entry
3. Link to technique_id from catalog

## Copyright Considerations

### Safe to Use
- Official instructor channels
- Publicly available on YouTube
- Educational/instructional content
- Embedding (not downloading)

### Avoid
- Pirated instructional content
- Reuploaded copyrighted material
- Private/unlisted videos
- Geo-restricted content

### Best Practice
- Link to original source
- Credit instructor in metadata
- Use official channel when available
- Do not host video files
