# Motion Developer - Animation Library

A React animation library with 30+ pre-built, customizable animations using Framer Motion. Includes Storybook documentation.

## Features

- 30+ animations (entrances, exits, attention, feedback)
- Storybook documentation
- Customizable timing and easing
- Copy-paste ready components

## Tech Stack

- React 18+ with TypeScript
- Framer Motion
- Storybook 8
- CSS Modules
- Vite

## Getting Started

```bash
cd motion-glossary
npm install
npm run dev
```

## Storybook

```bash
npm run storybook
```
