## Tips

1. **Keep it subtle** - Most UI animations should be 0.2-0.5 seconds
2. **Match the action** - Errors shake, success checks, loading spins
3. **Be consistent** - Use the same entrance animation throughout your app
4. **Respect users** - All animations disable for `prefers-reduced-motion`
