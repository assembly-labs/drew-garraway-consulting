## Timing Settings

### duration
**What it does:** How long the animation takes from start to finish.

**Think of it like:** A video playback speed. Lower = faster animation, higher = slower.

<table>
  <thead>
    <tr>
      <th>Value</th>
      <th>Effect</th>
      <th>When to use</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>0.1 - 0.2s</code></td>
      <td>Very fast</td>
      <td>Tiny interactions (hovers, clicks)</td>
    </tr>
    <tr>
      <td><code>0.3 - 0.5s</code></td>
      <td>Normal</td>
      <td>Most UI animations (buttons, cards)</td>
    </tr>
    <tr>
      <td><code>0.6 - 1.0s</code></td>
      <td>Slow</td>
      <td>Content reveals, modals appearing</td>
    </tr>
    <tr>
      <td><code>1.0s+</code></td>
      <td>Very slow</td>
      <td>Dramatic hero sections, backgrounds</td>
    </tr>
  </tbody>
</table>

**Try it:** Set duration to `0.2` then `1.5` - see how different it feels!

---

### delay
**What it does:** How long to wait before the animation starts.

**Think of it like:** A countdown timer before action begins.

<table>
  <thead>
    <tr>
      <th>Value</th>
      <th>Effect</th>
      <th>When to use</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>0s</code></td>
      <td>Immediate</td>
      <td>Most interactions</td>
    </tr>
    <tr>
      <td><code>0.1 - 0.3s</code></td>
      <td>Brief pause</td>
      <td>Second element in a sequence</td>
    </tr>
    <tr>
      <td><code>0.5s+</code></td>
      <td>Noticeable wait</td>
      <td>Loading states, staged reveals</td>
    </tr>
  </tbody>
</table>

**Pro tip:** Use delays to create sequences - first item: 0s, second: 0.1s, third: 0.2s

**Think of it like:** The length of a jump - small hop vs long leap.

## Scale Settings

### scale
**What it does:** How much bigger or smaller something gets.

**The number is a multiplier:**
- `1.0` = normal size (100%)
- `1.1` = 10% bigger (110%)
- `0.9` = 10% smaller (90%)

## Easing Settings

### easing / ease
**What it does:** Controls the "feel" of the animation - does it speed up? Slow down? Bounce?

**Think of it like:** A car's acceleration. Does it zoom off fast and brake slowly? Or start gentle and speed up?

<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Feel</th>
      <th>Best for</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>ease-out</code></td>
      <td>Fast start, gentle stop</td>
      <td>Most UI (feels responsive)</td>
    </tr>
    <tr>
      <td><code>ease-in</code></td>
      <td>Gentle start, fast end</td>
      <td>Exits, closing things</td>
    </tr>
    <tr>
      <td><code>ease-in-out</code></td>
      <td>Gentle start and end</td>
      <td>Looping animations</td>
    </tr>
    <tr>
      <td><code>linear</code></td>
      <td>Constant speed</td>
      <td>Progress bars, spinners</td>
    </tr>
  </tbody>
</table>

**The numbers `[0.4, 0, 0.2, 1]`:** This is a "cubic bezier" - think of it as a custom easing recipe. The default values work great for most cases!

## Color Settings

### colors (for gradients)
**What it does:** The colors used in a gradient animation.

**Format:** An array of color codes:
```javascript
colors: ['#ff0000', '#00ff00', '#0000ff']
//        red        green       blue
```

**Color code formats:**
- Hex: `#ff0000` (red)
- RGB: `rgb(255, 0, 0)` (red)
- Named: `red`, `blue`, `purple`

**Tip:** Use 3-4 colors that blend well. Try similar hues for smooth gradients.

---

### color / rippleColor
**What it does:** A single color for the effect.

**For ripples:** Use semi-transparent colors:
```javascript
rippleColor: 'rgba(255, 255, 255, 0.3)'
//           red  green blue  opacity (30%)
```

## Common Mistakes to Avoid

1. **Too fast:** Duration under 0.1s is hard to see
2. **Too slow:** Duration over 2s feels sluggish for UI
3. **Too bouncy:** Low damping (under 10) can feel unprofessional
4. **Too much movement:** Distance over 200px can be jarring
5. **Too much scale:** Scale over 1.5 often looks wrong

---

## Recommended Starting Points

### Button click feedback
```
scale: 0.95, duration: 0.1
```

### Card appearing
```
duration: 0.4, distance: 20, direction: 'up'
```

### Modal opening
```
duration: 0.3, scale: starts at 0.95
```

### List items
```
staggerDelay: 0.08, duration: 0.3, distance: 15
```

### Loading spinner
```
duration: 1, repeat: infinite
```

## Still Confused?

1. **Play with the controls!** Storybook lets you adjust values in real-time
2. **Start with defaults** - they're chosen for a reason
3. **Make small changes** - adjust by 0.1 at a time
4. **Trust your eyes** - if it looks good, it probably is good
