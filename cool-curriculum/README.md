# Cool Curriculum - AI Lesson Planner

AI-powered lesson material generator for K-4 educators. Creates differentiated, standards-aligned worksheets, quizzes, and activities using Claude AI.

## Features

- Generate lesson materials from prompts
- Standards-aligned content (K-4)
- Differentiated instruction support
- Multiple activity types (worksheets, quizzes, activities)

## Tech Stack

- React 19 + TypeScript
- Tailwind CSS
- Anthropic Claude API
- Express proxy server
- localStorage

## Getting Started

```bash
cd ai-lesson-planner
npm install
npm run dev
```

## Architecture

Uses a proxy server pattern for Claude API calls. Visual generation is currently mocked - needs backend for production.
