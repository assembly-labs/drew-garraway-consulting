    🐛 IF YOU SEE ERRORS
═══════════════════════════════════════════════════════════════════════════

"Permission denied" when running ./RUN_ME_FIRST.sh:
  → Run: chmod +x RUN_ME_FIRST.sh
  → Then try again

"mkcert: command not found":
  → The script will install it automatically
  → If it doesn't, run: brew install mkcert

"This connection is not private" on iPhone:
  → This is NORMAL!
  → Tap "Advanced" → "Proceed"
  → It's a self-signed certificate for local testing

File picker doesn't open on iPhone:
  → Check URL has 🔒 lock icon
  → Make sure both devices on same WiFi
  → Restart the server
