    ⚠️ COMMON ISSUES & FIXES
═══════════════════════════════════════════════════════════════════════════

"Permission denied" when running script:
  → Run: chmod +x RUN_ME_FIRST.sh

"mkcert not found":
  → Run: brew install mkcert

"This connection is not private" on iPhone:
  → This is NORMAL! Tap "Advanced" → "Proceed"
  → It's a self-signed certificate for local testing

File picker doesn't open on iPhone:
  → Make sure you see 🔒 lock icon in Safari
  → Make sure URL starts with https:// not http://

Can't connect from iPhone:
  → Both devices must be on same WiFi
  → Check firewall isn't blocking port 8443
