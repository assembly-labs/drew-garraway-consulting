# Photography & Visual Design Expert

A fast UI for sourcing images, icons, and visual design assets.

## Quick Start

```bash
npm install
npm start
```

Then open http://localhost:3000

## Features

- Search for photos, icons, illustrations, and vectors
- Quick access to popular design resources
- Clean, modern interface
- Ready to extend with API integrations

## Asset Sources

- **Unsplash**: Free high-resolution photos
- **Pexels**: Free stock photos & videos
- **Iconify**: 100k+ open source icons
- **Flaticon**: Vector icons & stickers
