# Printing from Public Computers

## After clicking "Print" or Control+P, follow these steps:

### Print Screen
1. Select "Cloud Print" (in red circle below)
2. Click "Print" (in green circle below)

[Screenshot showing print dialog with Cloud Print option circled in red and Print button circled in green]

### Print Confirmation
3. Make a note of the User ID and Password tapped to the bottom of the monitor
4. Go over to the Public Printer and follow the steps to release your pages.

[Screenshot showing Current Balance, Welcome screen with User ID and Password fields, and Color/Mono options]

[Screenshot showing document selection interface]

## Releasing a Print Job

### 1. Touch Elevate Sky Print
[Screenshot showing login screen with "Log in using your User ID and Password" prompt and circled green button]

### 2. Enter the User ID and password from bottom of monitor, then touch the green "Go" button.
[Screenshot showing login screen with password field circled in green]

### 3. Touch the icon that has a number in a white circle.
[Screenshot showing interface with Current Balance, Mono, Color options and "Unlimited" status]

### 4. Touch the name of the document and then touch the printer icon.
[Screenshot showing document selection interface with printer icon circled]
