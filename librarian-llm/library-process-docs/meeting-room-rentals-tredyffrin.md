# Interested in renting a Meeting Room at the Tredyffrin Public Library?

Follow these steps:

1. Go to www.tredyffrinlibraries.org
2. Click "About"
3. Click "Meeting Room Rentals"
4. Click the link titled Tredyffrin Township Libraries Meeting Room Policy to read the current policy.
5. Complete and submit the Room Rental Request on the website.
