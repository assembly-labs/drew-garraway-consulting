SCOUT LIBRARIAN COHORT RECRUITMENT MATERIALS
============================================
Last Updated: December 3, 2024

WHAT'S INCLUDED
---------------
This package contains three HTML files for recruiting libraries to join Scout's founding cohort:

1. librarian-cohort-generic.html
   - General recruitment page for broad outreach
   - Can be posted on website or sent to multiple libraries
   - Self-contained, no external dependencies

2. librarian-cohort-template.html
   - Customizable template for personalized outreach
   - Contains {{VARIABLES}} to replace with library-specific information
   - Includes instruction box at top (remove before sending)
   - Use this to create personalized versions for specific libraries

3. librarian-cohort-tredyffrin.html
   - Example of personalized version for Tredyffrin Public Library
   - Shows how to customize the template with specific details
   - Ready to send as-is to Tredyffrin

HOW TO USE
----------
For General Outreach:
- Use librarian-cohort-generic.html
- Can be hosted on website or attached to emails
- Works offline, no internet required

For Personalized Outreach:
1. Open librarian-cohort-template.html
2. Replace all {{VARIABLES}} with specific library information:
   - {{LIBRARY_NAME}} - Full library name
   - {{LIBRARY_SHORT}} - Short name/acronym
   - {{COMMUNITY_DESC}} - Community description
   - {{PATRON_COUNT}} - Number of patrons served
   - {{UNIQUE_FEATURE}} - Something notable about the library
   - {{ILS_SYSTEM}} - Their ILS platform
   - {{DIRECTOR_NAME}} - Library Director's name
   - {{SPECIFIC_CHALLENGE}} - Known pain point
   - {{WHY_SELECTED}} - Why this library specifically
3. Remove the yellow instruction box at the top
4. Save with library-specific filename
5. Send directly via email or host online

VIEWING THE FILES
-----------------
These are standalone HTML files that work in any web browser:
- Double-click to open in your default browser
- Or right-click and "Open with" your preferred browser
- No internet connection required
- Works on Windows, Mac, Linux, tablets, phones

CUSTOMIZATION TIPS
------------------
- Research the library before personalizing (website, annual reports, news)
- Use specific examples relevant to their community
- Reference actual programs or services they offer
- Mention regional conferences they might attend (PaLA, NJLA, etc.)
- Include local landmarks or community characteristics
